# Microsoft Python Developer Certificate syllabus

### Certificate overview

#### Ready to turn your Python passion into a fulfilling career? The Microsoft Python Developer Certificate is your ticket to success. Whether you're an enthusiastic hobbyist or have a year of Python under your belt, this comprehensive program will elevate your Python skills, expand your knowledge of the many different applications of Python, and equip you with both the practical experience and work samples employers crave. Upon successful completion, you'll earn a certificate that validates your Python proficiency and opens doors to exciting opportunities in the tech industry.
